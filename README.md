## ğŸ“– Tentang Proyek

Repositori ini mencakup implementasi berbagai metode numerik dan simulasi fisika yang dipelajari selama perkuliahan. Fokus utama meliputi penyelesaian persamaan diferensial, integrasi numerik, pencarian akar persamaan, serta simulasi sistem fisis menggunakan bahasa **MATLAB** dan **C++**.

Topik yang dibahas meliputi (namun tidak terbatas pada):
* Metode Numerik Dasar (Akar Persamaan, Integrasi, Diferensiasi)
* Penyelesaian Persamaan Diferensial Biasa (ODE)
* Simulasi Dinamika Partikel
* Proyek Akhir Pemrograman C++

## ğŸ“‚ Struktur Repositori

Direktori ini diatur berdasarkan mata kuliah dan tahun ajaran:

### 1. `Komputasi Fisika I 2024`
Folder ini berisi materi dan kode untuk mata kuliah Komputasi Fisika I.
* **Fokus:** Dasar-dasar metode numerik dan algoritma komputasi.
* **Bahasa:** MATLAB (Dominan).

### 2. `Komputasi Fisika II 2025`
Folder ini berisi materi lanjutan untuk Komputasi Fisika II.
* **Fokus:** Pemodelan sistem fisika yang lebih kompleks dan simulasi lanjut.
* **Bahasa:** MATLAB.

### 3. `Pemrograman C++ 2024`
Folder ini didedikasikan untuk mata kuliah Pengantar Pemrograman.
* **Fokus:** Implementasi logika pemrograman terstruktur/berorientasi objek (OOP) dalam konteks fisika atau aplikasi umum.
* **Bahasa:** C++.

## ğŸ›  Prasyarat

Untuk menjalankan kode-kode di dalam repositori ini, pastikan Anda telah menginstal perangkat lunak berikut:

* **MATLAB** (Untuk file `.m`) - [Link Download](https://www.mathworks.com/products/matlab.html)
* **C++ Compiler** (Dev-C++, GCC, Clang, atau MSVC untuk file `.cpp`)
    * Pengguna Windows dapat menggunakan [MinGW](https://www.mingw-w64.org/) atau Visual Studio.
    * Pengguna Linux/Mac biasanya sudah memiliki `g++` atau `clang`.

## ğŸš€ Cara Penggunaan

1.  **Clone Repositori**
    ```bash
    git clone [https://github.com/akbar-kurnia1/pemrograman-dan-komputasi-fisika.git](https://github.com/akbar-kurnia1/pemrograman-dan-komputasi-fisika.git)
    ```
2.  **Masuk ke Direktori**
    ```bash
    cd pemrograman-dan-komputasi-fisika
    ```
3.  **Menjalankan File MATLAB**
    * Buka MATLAB.
    * Navigasikan ke folder `Komputasi Fisika I 2024` atau `Komputasi Fisika II 2025`.
    * Jalankan script yang diinginkan melalui *Command Window* atau tekan **Run**.

4.  **Menjalankan File C++**
    * Navigasikan ke folder proyek C++.
    * Compile file C++:
        ```bash
        g++ nama_file.cpp -o output
        ```
    * Jalankan program:
        ```bash
        ./output  # (Linux/Mac)
        output.exe # (Windows)
        ```
